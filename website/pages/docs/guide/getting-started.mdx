# Setting up Rust

The easiest way to install Rust is to use the [rustup](https://rustup.rs/) tool.

# Cargo

Add Arcon as a dependency in your ``Cargo.toml``. Currently arcon does not have a stable version on crates.io, so 
we add a git dependency instead. Note that we also include [prost](https://github.com/tokio-rs/prost) 
as it is required when declaring more complex Arcon data types.

```rust
[dependencies]
arcon = { git = "https://github.com/cda-group/arcon" }
prost = "0.7"
```

# Creating an Arcon Application
In ``src/main.rs``

```rust
use arcon::prelude::*;

fn main() {
    let mut app = Application::default()
        .with_debug_node()
        .iterator(0..100, |conf| {
            conf.set_arcon_time(ArconTime::Process);
        })
        .filter(|x| *x > 50)
        .map(|x| x + 10)
        .build();

    app.start();

    std::thread::sleep(std::time::Duration::from_millis(2000));
    let debug_node = app.get_debug_node::<i32>().unwrap();

    // Access the data within the Debug Node and verify the result
    debug_node.on_definition(|cd| {
        let sum: i32 = cd.data.iter().map(|elem| elem.data).sum();
        assert_eq!(sum, 4165);
    });
}
```
